Features

  • Versatile BLOB handling

    • Http transmission of BLOBs for Unified API applications.
    • The Content Server automatically chooses between different BLOB stores.
    • File-based BLOB storage.
    • Storing BLOBs in different database tables.

  • API to use different search engines with the CAE Feeder.

  • Analyze the document types definition and build editing forms for Editing Services for JSF.

  • Type safe Unified API.

  • Contributions:

    • CAE Console: development tools for better insight to the CAE

    • JCR Level 1 implementation (excl. query interface)
