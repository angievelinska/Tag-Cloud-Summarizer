DOUBLE

java.lang.Double

Due to the mapping to the underlying database column type, not all valid java.lang.Double values may be stored in a DOUBLE property. Instead, the valid DOUBLE values depend on the chosen database system. On installations based on an Oracle relational database, the DOUBLE property type is mapped to a DOUBLE PRECISION column type with a smaller range of valid values. For instance, the value java.lang.Double#MIN_VALUE (2^-1074) cannot be stored in this case.
