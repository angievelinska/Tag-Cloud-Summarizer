After configuring the database, you can test your setup by starting your webapp:

  1. Start the Tomcat installed as part of the CAE web application template.
  2. Navigate your browser to http://localhost:8001/contentapplicationengine-template/servlet/
  3. Congratulations! If you see the starting page (below) listing the configured profile card types, your setup is correct and you can start exploring the Social Software Extension deliverable by extending the minimal web application and using the documented API.
