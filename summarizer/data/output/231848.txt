

The RichText component provides enhanced WYSIWYG possibilities for text formatting. The document window is organized in three parts: 

The menu bar with access to all functions (1)


The tool bars with quick access to the most important functions. 

The tool bar with functions applicable to the document (2)
The tool bar with the functions applicable to the version you work on independent from the current field (3).
The tool bar with the functions applicable to the current field you work on (4)
The working area for text input (5)
