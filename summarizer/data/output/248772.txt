

The CoreMedia Delivery Developer Manual describes how to develop content delivery solutions which use the different CoreMedia components.

In Section 2 you will find a short overview of the CoreMedia system.
In Section 3 you will learn how to deliver content using the CoreMedia Active Delivery Server.
In Section 4 you will learn how to use the ProActive Delivery Server for development.
In Section 7 you will learn how to use Web Services for your CoreMedia applications.
In Section 5 you will find an example which describes how to create content and how to write templates for the ADS.
