

<wage:toolbar>

A WAGE toolbar can contain one or more WAGE command buttons, possibly grouped by WAGE separators. See XXX for an example on how to use a WAGE toolbar. Attributes:

focusManager (required): id of the FocusManager that will be used to find the focused editor for calling commands defined with a <wage:cmdButton> tag.
styleClass (optional): style class to use for the toolbar
large (optional): true or false, use small or large icons
highlightStyleClass (optional): style class to use for highlighting
