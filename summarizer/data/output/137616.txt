

This chapter describes the installation of the CoreMedia Web Editing Server. You can install:

The WAGE Webapplication
The WebEditor Webapplication
The WebDAV Webapplication

Proceed as follows:


Select the CoreMedia WES  option and click [Next].

The next window opens up and shows the components of the Web Editing Server to install. You can select the WAGE API (WAGE Webapplication), the CoreMedia WebEditor (WebEditor Webapplication) and/or the CoreMedia WebDAV support (WebDAV Webapplication).
