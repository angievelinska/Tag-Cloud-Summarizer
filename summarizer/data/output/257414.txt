

Starting workflows

The start of a workflow in the WebEditor differs slightly from the start in the CoreMedia Editor because an additional window is involved. So, the start will be described in this chapter. For all other workflow tasks, read Kapitel.



Overview window


Choose File | New workflow... and select the workflow (see illustration.)


Click [Start Workflow].



Workflow window


Choose Window | Workflow window, then choose New workflow ... and select one.


Click [Start Workflow].
