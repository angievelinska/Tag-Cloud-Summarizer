

<wage:menuBar>

A WAGE menuBar can be used to render a drop down menu bar containing one or more WAGE menus, which in turn contain menuItems grouped by menuSeparators. Attributes:

styleId (required): an id for the menu
styleClass (optional): style class to be used for the menu bar
menuStyleClass (optional): style class for the menus on the menu bar
selectedMenuStyleClass (optional): style class for a selected menu
focusedMenuStyleClass (optional): style class for a focused menu
