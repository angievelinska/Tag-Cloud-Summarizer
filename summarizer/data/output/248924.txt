

The CoreMedia SCI allows to check user rights on resources (see Section). The access control used in the Menu Site example should enable the following scenario:

Each user is able to see the dishes of the day.
Each user is able to see all fish dishes.
Members of the group "PoultryFan" can only see the poultry dishes.

Two steps are necessary to use access control:
Enable access control in the config.bsh file of the ADS.
Create users and groups and add appropriate rights to the resources.
