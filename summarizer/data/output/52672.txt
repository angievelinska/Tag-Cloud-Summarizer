

<wage:css>

Renders a stylesheet link to the stylesheet set with the css attribute. If the css attribute isn't set, a link to the wage.css stylesheet is rendered, which includes the styles for WAGE toolbars, etc. For displaying a XML property with the coremedia-richtext-1.0 grammar you should use this tag with the css attribute set to "coremedia-richtext-1.0". Attributes:

css (optional): name of the stylesheet for which a link should be rendered, the stylesheet has to be placed in the css directory of your web application
uriPrefix (optional): uri to the context containing the css directory, if not set generator.getContextPath() will be used
