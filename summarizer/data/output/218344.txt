

How to add properties depends on the property type.

For properties of the type,

XmlProperty
BlobProperty
LinkListProperty

proceed as follows:
Add the new property types to the document types definition file.
Restart the Content Server.

For properties of all other types proceed as follows:
Stop the Content Server.
Change the document types definition file.
Be sure, that the properties sql.schema.checkColumns and sql.schema.alterTable in the <ContentServerHome>/properties/corem/sql.properties file are set to "true"
Restart the Content Server.
