

Element XPathField

The element XPathField specifies an additional field to be indexed with the content of a fragment of an XML property. The fragment is selected with an XPath expression, which is specified in the subelement XPath. For each indexed document the first matching Property subelement defines the XML property that is used as source for the element. The field will be empty if there is no matching property.
