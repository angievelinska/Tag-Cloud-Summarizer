

Creating internal links

In order to designate a selected text section as an internal link proceed as follows:


Mark the text in the document which should appear as a link.




Designate the marked text as an internal link with the following actions.



Select Format|Internal Link from the menu bar or Internal Link from the Context menu.


Click  and  on the tool bar.


Press the keyboard shortcut <Ctrl>+<Shift>+<i>.
