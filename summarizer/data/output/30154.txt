Description 

class 

This attribute defines the class which manages the startup of the workflow. Own classes must implement the interface hox.corem.editor.workflow.WorkflowStartup.

As a default, the class hox.corem.editor.generic.GenericWorkflowStartup is used. It sets the ressource variable with the selected resource(s) and opens a window for setting the other workflow variables.
