

The CoreMedia Web Editor Server contains a WebDAV server which supports access to content of the CoreMedia Content Management Server with WebDAV-enabled applications like Microsoft Word 2000.

The WebDAV support is provided by a web application which is deployed in the servlet container. The contents of the Content Management Server appear in a web folder of the WebDAV client. Section describes how to create a Windows web folder. Configure WebDAV support in the files

webdav.properties,
capclient.properties
log.properties (to configure logging, see Section)

located in the directory <WebDAVServletContext>/WEB-INF/properties/corem where <WebDAVServletContext> is the path to the WebDAV servlet context, for example  jakarta-tomcat/webapps/webdav.
