4#
If you want to set up Tomcat now, click [Yes]. If you want to configure later manually, click [No].

 If you have clicked [Yes] the next window will open up in which you can configure the server.xml file. In general, no configuration is necessary in this file. If you use more than one Tomcat on the same machine you might have to adjust the Shutdown port.
5#
Enter the ports you want to use and click [Next].

The last window will open up and will approve the success of the installation.
6#
Click [Finish] to finish the installation.
