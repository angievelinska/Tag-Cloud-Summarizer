

The CoreMedia Lightweight Portal extends a standard CAE Web Application with the ability to display and interact with portlets.

As defined in the Java Portlet Specification, a portlet is contained in a portlet application, which is deployed in a portal. In the case of CoreMedia, the role of the portal is implemented by four interacting components:

the Tomcat servlet engine


the CAE Portlet Container, consisting of

the CAE Portlet Container Adapter
the portlet container
the CAE Web Application
