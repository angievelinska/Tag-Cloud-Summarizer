You can explicitly register a builder to generate values for keys of any scheme you like. However you should only use URI schemes that start with a custom prefix like your domain. Builders can be configured using a Builder element with attributes for scheme and builder class below the Pads element. A Builder element makes it also possible to pass configuration settings to your builder. You don't need a Builder element if your builder doesn't require any configuration and only generates values for keys with the builder's fully qualified class name.
