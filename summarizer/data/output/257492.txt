

The Query window allows to search for documents. The window consists of two main parts below the menu bar and toolbar.


(1) Menu bar


The menu bar provides access to all features, handling of the query itself, the conditions, view and navigation options and access to other windows.


(2) Toolbar 


Provides access to all funcions except view and window options.


(3) Query window 


The Query tab in the query window lets you select one of the available doctypes, limit the hits to 25, 50,100, or all and displays the number of found documents. The Comment tab (4) is described in section.


(5) List of chosen and set conditions


They can be displayed in a tree view via View | Show tree, too.


(6) Result window


Displays the search result with configurable view options as well as the time the query took in its footer.
