

In the dialog box only documents that match the allowed document types of the link list are shown. You can search for a specific document using the search field.

If the document exists, proceed as follows:
3#
Select the document for pasting from the folder hierarchy.
Confirm by clicking on [Choose].

If you want to create a new document perform the following steps:
3#
Choose the folder in which the new document should be created and open the context menu with the right mouse button.
Choose the appropriate document type in the menu item New resource...


Enter the name of the document in the opening dialog and click [Create].


Select the newly created document in the selection dialog and click [Choose].
