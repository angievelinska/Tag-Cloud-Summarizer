Creating external links

To create an external link in your document, carry out the following steps:


Mark the text in the document which should appear as a link.


Designate the marked text as an external link:



Select Format|Link from the menu bar.


Select Link from the Context menu.


Click  and  on the toolbar..


Press the keyboard shortcut <Ctrl>+<Shift>+<v>.

A dialog box appears for creating the external link.
