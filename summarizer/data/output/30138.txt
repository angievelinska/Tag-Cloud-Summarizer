class 

With this attribute the class used for determining the customized dictionary used in the spellchecker is configured. As default the class hox.corem.editor.spellchecker.Dictionary will be used. This class uses two customized dictionary created in the folder tree of the CoreMedia System. For usage of these dictionaries see the User Manual.

For own classes, the interface com.coremedia.spellchecker.CustomDictionary must be implemented.
