


 

Attribute
 

Optional
 

Description
 

folder
 

No
 

The folder below which resources reside. If the type includes folders, a value for the folder itself is also generated. 
 

type
 

No
 

The resource type to generate values for. This can be the document type (e.g. "Article") or the special values "Folder" to generate values for folders and "*" to generate values for all resources.
 

uri
 

No
 

A key pattern with the placeholder "{0}" which will be replaced with the resource id for each selected resource.
