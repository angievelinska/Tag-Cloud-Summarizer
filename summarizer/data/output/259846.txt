

You can select one of three installation options:

Starter Kit Standard Installation installs a complete CMS with a Starter Kit. 
Starter Kit Custom Installation installs the CMS and Starter Kit components you have chosen. 
Starter Kit Files-Only installs the files of the Starter Kit without any CMS components. 
3#
Select the option you want to use and click [Next].
