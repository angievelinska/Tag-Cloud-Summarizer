

  

Attribute 

Description 

debug|error|info 

Defines the log level "debug", "error" or "info". Value must be "true". 

message 

The message which is printed to the log. 

facility 

Define an own log name for the output. If you use this attribute you can define an own log target for the ouput in the workflowserver.properties file e.g.:

log.action.3.class=FileAction

log.action.3.selectors=workflow.<MyLogName>:info

log.action.3.initArgs=
