

Logging out

In order to log out from the dashboard and to terminate your session, use the "logout" link in the upper right corner next the user's login name.

Opening the Analyzed Page

Using the symbol "eye", you can open the page for which reports are shown in a separate window. This option is only available if configured appropriately by the administrator.

Open the Dashboard

Using the symbol "pie chart", you can open the full Analytics Dashboard in a separate window.
