

A property language resolver factory  is used to determine the language of a property which will be used for the spellchecker.

Interface to implement

Own property language resolver factories must implement the interface hox.corem.editor.PropertyLanguageResolverFactory with the method getLanguageResolver. In addition an inner class must be created which implements the interface LanguageResolver (see example).
