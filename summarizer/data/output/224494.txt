

 To create a new document:
Select the folder in which the new document should be created.
There are the following possibilities of creating a document:

Select File|New|Document type from the menu bar.
Select New|Document type from the Context menu.
Click  on the toolbar for creating new documents.

A dialog box appears, asking you to enter a name, tpye, folder for the new document. In addition, you can choose to open the document immediately.
